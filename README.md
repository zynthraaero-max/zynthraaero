# zynthraaero
la aplicación creada y dada en este espacio es con el fin de acelerar y crear procesos estandarizados para el manejo de documentos 
aeronauticos permitiendo agilidad, dismunuyendo errores, realizando mejor proceso de comunicación con el cliente y proveedores,gestion inteligente de herramientas.

La aplicación se basa en un buscador vinculado a los archivos excel dados en la empresa con su información lo que permite visualizar

Pasos a seguir para manejar la aplicacion
1. Preparación del Entorno
Instalación: Descarga e instala Visual Studio Code en tu equipo.
Acceso alternativo: Si prefieres no realizar una instalación local, puedes acceder a la aplicación directamente mediante el [enlace o código fuente proporcionado en este repositorio].
3. Configuración de la Matriz de Datos
Es fundamental vincular la aplicación con tus archivos de origen para que el buscador funcione correctamente:
Localiza la ruta donde se encuentran tus archivos Excel (Matriz).
Dentro de la configuración de la aplicación, actualiza el parámetro de Ruta de la Matriz para que el sistema reconozca dónde buscar la información.
4. Definición del Directorio de Instalación
Selecciona la carpeta o unidad de red donde deseas que resida la aplicación. Asegúrate de tener permisos de lectura y escritura en dicha ubicación para evitar errores de sincronización.
5. Ajuste de Visualización
Una vez vinculada la matriz, la aplicación está lista para su uso.
Si tu rol es únicamente de consulta (visualización), puedes ajustar el formato de salida en la configuración del panel de control. Esto permite filtrar las columnas relevantes y ordenar los datos para una lectura más cómoda.
